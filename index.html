Interesting Pins Just for Kids... and Parents

Overview

This is a purely front end build out of all the pages, modals, and states needed to create KidPins core site. Includes html, css, and javascript. Will later convert to Ruby on Rails using Haml/Sass.

This site is basically Pinterest for parents interested in great finds for to help their children. KidPins uses the concept of 4 buckets of Pins; Article, Gift, Idea and General Pin types. KidPins also has a very small taxonomy applied which creates top level categories to be used in search and general board browsing.

Will be monetized through the use of skimlinks.com API product feed in order to supply an onsite store with products.

Third Party Usage

base html is HTML5 Boilerplate v 4.0.2.

base javascript is jquery.js v 1.8.2. Download it here.

Modal javascript is lightview.js v 3.2.7.

Pinterest style box stacking javascript is jquery.Masonry.js v2.1.06. Great supporting articles: Three Super Easy Ways to Pull Off a Masonry Layout, and Get started with jQuery Masonry.

Plan to include responsive design, hand rolled, with media queries. Need to research pixel width break points for mobile devices and tablets. May base this off of twitter bootstrap for reference.
